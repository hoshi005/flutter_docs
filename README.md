## 使い方

`npm` を利用して `honkit` をインストール

```bash
$ npm install honkit --save-dev
```

`honkit init` コマンドで初期化

```bash
# 作りたいドキュメントのディレクトリで実行
$ npx honkit init
```

`honkit serve` コマンドでプレビューできる

```bash
$ npx honkit serve
```

`honkit build` で `_book` にビルド結果のHTMLを出力

```bash
$ npx honkit build
```
